盒子模型(box model)

作用: 封装周围的HTML元素

一般都是在 <div>(大盒子) 和 <span>(小盒子) 中写

  • 这两个的样式都是空, margin, border, padding, content 都是0
  • 区别: <div> 独占一行, <span> 可一行多个

关系

  • margin 是外边距
  • border 是边框
  • padding 是内边距
  • content 是文本内容

实际上盒子的大小 = content + pddding + border + margin

border

没设置默认为0

outline

outline 不占盒子空间, 只是起到对文本的修饰作用
